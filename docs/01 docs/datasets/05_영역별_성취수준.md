1. **영역별 성취수준(DomainAchievementLevels)**:

    - 수와 연산 영역에 대해 A부터 E까지의 성취수준을 내용 범주별(지식·이해, 과정·기능, 가치·태도)로 구분하여 총 15개의 항목으로 구성
    - 각 항목은 고유 ID, 영역 ID, 수준, 범주, 내용을 포함

2. **영역(Domain)**:

    - 수와 연산 영역의 기본 정보(ID, 이름, 코드, 설명)

3. **성취수준(AchievementLevels)**:

    - A부터 E까지의 5단계 성취수준 정의

4. **내용 범주(ContentCategories)**:
    - 지식·이해(KU), 과정·기능(PF), 가치·태도(VA) 세 가지 범주 정의

이 JSON 데이터를 Neo4j에 로드하면 다음과 같은 관계가 형성됩니다:

-   (DomainAchievementLevel)-[:BELONGS_TO_DOMAIN]->(Domain)
-   (DomainAchievementLevel)-[:HAS_LEVEL]->(AchievementLevel)
-   (DomainAchievementLevel)-[:BELONGS_TO_CATEGORY]->(ContentCategory)

이 데이터를 통해 영역별로 어떤 수준(A~E)에서 어떤 범주(지식·이해, 과정·기능, 가치·태도)의 성취를 기대하는지 체계적으로 관리하고 활용할 수 있습니다.
